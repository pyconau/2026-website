Adarsh Divakaran is a Python Developer Advocate at [SerpApi](https://serpapi.com/), where he helps developers build with search APIs and Python tooling. He has spoken at several international Python conferences, including PyCascades, EuroPython, PyCon Japan, DjangoCon, and FlaskCon.

He is the author of Deep Dive Python, published by Apress, and a Microsoft MVP in Python. He is also a co-organizer of [Trivandrum Python Community](https://trivandrumpython.github.io/), a Python user group in Kerala, India.
