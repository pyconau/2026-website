Learn about learning, teach about teaching, and educate about education! 

Do you code with kids? Tinker on educational teachings? Has your Python programming in schools gone fabulously, or terribly? This track is for you!

Python programming is becoming more common in education settings, whether taught as part of the Digital Technologies classrooms, integrated across the curriculum, or embedded in coding clubs. We'll hear from teachers, education professionals, edutech enthusiasts and more about what's worked, what hasn't and everything in-between!
